Library Overview

The ProtoSynthesis Library is the structured knowledge repository within the wider ProtoSynthesis architecture.

It provides organised access to books, articles, whitepapers, research materials, and other knowledge artefacts indexed through a unified registry system.

The Architecture defines systems and structural rules. The Library contains the knowledge produced within those systems.
