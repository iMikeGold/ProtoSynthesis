Overview

G01DN is the structural governance system of ProtoSynthesis. It defines how authority flows, how systems are structured, and how validity is determined across all architectural layers.

Layer Model

  • L0: Constitutional Form — existence rules
  • L1: Protocol Form — interaction rules
  • L2: Standards Form — correctness rules
  • L3: Reference Form — representation rules

Core Principle

Authority flows strictly downward: L0 → L1 → L2 → L3. Higher layers define constraints; lower layers must comply.
