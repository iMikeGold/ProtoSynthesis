What is L1?

The Protocol Form defines how system components interact within G01DN. It governs relationships, not existence.

Core Function

  • Defines interaction rules between layers
  • Controls inheritance behaviour
  • Manages conflict resolution flow

Principle

Protocol does not define what exists — only how existing structures interact.
